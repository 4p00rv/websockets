Open-Source Projects

Binary.com maintains a number of open-source projects on github. Feel free to fork these repositories, and if you wish to submit an enhancement to a project, submit a pull-request. You may use desktop.github.com if you are unfamiliar with git.

The source for this API documentation website itself is maintained at https://github.com/binary-com/websockets.

Binary.com is developing an open-source mobile application at https://github.com/binary-com/mobile as well as a next-generation user-interface which acts as both a mobile application and a desktop application at https://github.com/binary-com/binary-next-gen. A separate project is underway to build a desktop application at https://github.com/binary-com/webtrader.

The front-end code for the binary.com website itself is open-source at https://github.com/binary-com/binary-static.

Binary.com's open-source projects are typically published via the Github Pages facility.

Security

Binary.com invests considerable time and effort into website security. Should you encounter what you believe to be any security issue whilst coding against the API, please report it on https://hackerone.com/binary.

Employment opportunities

Binary.com is expanding rapidly and is constantly on the lookout for talented software engineers to join our team. The best way to pre-qualify for a job is to submit a useful pull-request to any of our open-source projects. Please find out more about working for binary.com at https://www.binary.com/careers.
