Open-Source Projects

We maintain a number of open-source projects on GitHub. Feel free to fork these repositories, and if you wish to submit an enhancement to a project, submit a pull request. You may use GitHub Desktop if you are unfamiliar with git.

The source for this API documentation website is maintained at github.com/binary-com/websockets

Binary.com is actively developing open-source applications:

  • Binary Mobile – a mobile app for Android and iOS
  • Binary Next-Gen – a next-generation UI; both a mobile application and a desktop application
  • WebTrader – a desktop application

The front-end code for the Binary.com website itself is open-source at github.com/binary-com/binary-static

Binary.com's open-source projects are typically published via the Github Pages service.
